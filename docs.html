üéØ The Growth Thesis

Problem

Vity's core value - "Your AI remembers you" - is invisible until experienced. Users don't know what they're missing until context injection saves them from repeating themselves.

Opportunity

The moment Vity successfully recalls context is a peak emotional moment. Users think: "Wow, it actually remembered!" This is the highest-intent moment to ask for a referral.

Hypothesis: If we prompt users to share immediately after a successful context recall (detected via 3+ message exchanges), we can achieve a 5-15% referral click rate and 2-3% viral coefficient.

üî¨ Experiment Design

Trigger Logic

USER SENDS MESSAGE ‚Üí Vity injects context ‚Üí AI responds with personalization
                                                      ‚Üì
                              User realizes "it remembered me" (WOW MOMENT)
                                                      ‚Üì
                                    After 3rd message: Show referral prompt

Why 3 messages?

  • Message 1: User asks something
  • Message 2: AI responds with Vity-injected context
  • Message 3: User confirms value ("yes, that's right" or continues naturally)

This 3-message threshold correlates with successful context utilization.

Targeting Rules

Condition Action
First 3 messages in session Show modal
Already shown this session Suppress (localStorage)
User dismissed previously Wait 7 days before retry
User clicked "Get Link" Never show again (converted)

üõ°Ô∏è Privacy & Trust Considerations

Aligning with Gaurav's ZDR (Zero Data Retention) philosophy:

  1. No tracking without consent - All analytics are opt-in
